Rimuovere la parola “Collegamento”

Fonte trucchi:Amico PC

Questo trucco permette di aprire eliminare la parola collegamento a da un link
    
     HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer
     link=1e 00 00 00 a 00 00 00 00 ;bisogna riavviare

Cambiare proprietario di Windows

Questo trucco permette di cambiare amministrazione di sistema 
   
    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion
    RegisteredOwner ;dopo aver cambiato nome e riavviato il sistema digitare winver.exe per verificare

Eseguire CMD da qualsiasi cartella

    HKEY_CLASSES_ROOT\Drive\shell\cmd
    HKEY_CLASSES_ROOT\Drive\shell\cmd 
    Extended-BAK ;questo valore di default e' solo Extended
