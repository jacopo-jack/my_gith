Il menu a tendina.

Questo trucco permette di personalizzare le app che compaiono nel menu a tendina del Desktop
    
    HKEY_CLASSES_ROOT\Directory\Background\shell
    command key ;e' una nuova chiave che si crea col tasto destro del mouse
    command     ;la sotto chiave dove inserire l'intero percorso dell'applicazione che si vuole aggiungere al menu
